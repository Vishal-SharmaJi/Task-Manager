# Team Task Manager - Demo Script

## ⏱️ Duration: 3-5 Minutes

### 1. Introduction (30s)
- "Hello! This is Team Task Manager, a collaborative platform designed for modern teams to manage projects and track tasks efficiently."
- "The app is built using the MERN stack (MongoDB, Express, React, Node.js) with a focus on high-quality design and clean architecture."

### 2. Authentication & Roles (1m)
- "Users can sign up as either an Admin or a Member. Admins have full control over the workspace, while Members focus on their assigned tasks."
- "We use JWT for secure authentication and Bcrypt for password hashing. Notice the sleek, glassmorphism-inspired login and registration pages."

### 3. Admin Dashboard & Projects (1m)
- "As an Admin, I can see a dashboard with real-time statistics: total tasks, completed ones, and even overdue items."
- "In the Projects section, Admins can create new initiatives. Each project acts as a container for team members and their respective tasks."

### 4. Kanban Task Board (1m)
- "The heart of the app is the Project Details page. It features a Kanban-style board with 'Todo', 'In Progress', and 'Completed' columns."
- "Tasks include titles, descriptions, assignees, and deadlines. Members can quickly update the status of their tasks with a single click, triggering smooth animations powered by Framer Motion."

### 5. Technical Excellence (30s)
- "The code follows a modular structure, separating concerns into routes, controllers, and models. The frontend is built with Vite for speed and Tailwind CSS for a highly customized, responsive UI."
- "The entire application is deployment-ready for Railway, making it easy to scale from a prototype to a production environment."

### 6. Conclusion
- "Team Task Manager isn't just a tool; it's a foundation for better collaboration. Thank you!"
